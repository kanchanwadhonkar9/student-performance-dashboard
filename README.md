## 📊 Project Overview

Analyzed **300+ student records** using Python (Pandas) for data cleaning and duplicate removal. Built interactive Power BI dashboard to track pass/fail trends, subject-wise performance, and attendance correlation. Identified top 3 factors affecting scores and automated monthly reports — **reduced manual work by 60%**.

---

## 🎯 Key Achievements

- ✅ Cleaned and deduplicated 300+ student records
- ✅ Built interactive Power BI dashboard with real-time filtering
- ✅ Identified attendance as strongest predictor of student performance
- ✅ Automated monthly reporting workflow
- ✅ Reduced manual report generation time by 60%

## 📈 Key Insights

### Top 3 Factors Affecting Student Scores

1. **Attendance Rate** - Strong positive correlation (0.78)
2. **Study Hours** - Moderate positive correlation (0.65)
3. **Previous Semester Score** - Strong predictive factor (0.72)

## 🚀 Getting Started

### Prerequisites

- Python 3.8+
- Power BI Desktop
- Microsoft Excel

### Installation

```bash
# Clone the repository
git clone https://github.com/kanchanwadhonkar9/student-performance-dashboard.git

# Navigate to project directory
cd student-performance-dashboard

# Install dependencies
pip install -r requirements.txt
```

### Running the Analysis

```bash
# Step 1: Clean and prepare data
python code/data_cleaning.py

# Step 2: Run analysis and generate visualizations
python code/analysis.py

# Step 3: Generate monthly reports
python code/generate_report.py
```
